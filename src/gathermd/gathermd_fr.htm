Analyser les fichiers de données pour des informations sur les variables

Cette boîte de dialogue permet de créer un jeu de données contenant des informations sur les dictionnaires de variables dans un ou plusieurs fichiers IBM® SPSS® Statistics, SAS ou Stata.

►Pour exécuter cette procédure, dans les menus, choisissez :

 Fichiers
  Rassembler des informations sur les variables...

► Toutes les variables des fichiers sélectionnés sont incluses.

Répertoire de recherche : Indiquez le répertoire de recherche. La recherche est également effectuée dans ses sous-répertoires.

Modèle de nom de fichier : Un motif d'expression standard peut être utilisé pour filtrer les fichiers traités. Par exemple, voiture limite les fichiers à ceux dont le nom commence par "voiture". Le motif .*voiture accepte n'importe quel nom de fichier contenant le terme « voiture ».

• Les expressions standard ne sont pas les mêmes que les caractères génériques du nom de fichier qui se trouvent dans de nombreux systèmes d'exploitation. Par exemple, abc* trouera tous les noms commençant par ab : cela signifie littéralement ab suivi de zéro ou de plusieurs c.

• L'expression standard n'est pas sensible à la casse et est appliquée au nom du fichier sans extension.

Pour obtenir une explication complète des expressions régulières, voir http://www.amk.ca/python/howto/regex/. Si aucun motif n'est donné, tous les fichiers des types spécifiés sont traités.

Types de fichiers à rechercher : Cochez les cases correspondantes aux types de fichiers désirés. Les types de fichiers sont :

• IBM SPSS Statistics : .sav, .por

• SAS : .sas7bdat, .sd7, .sd2, .ssd01 et .xpt

• Stata : .dta

Liste d'attributs personnalisés à inclure : Entrez les noms de chaque attribut personnalisé dont les valeurs doivent être incluses dans la sortie.

Longueur maximale des valeurs d'attribut : La valeur des attributs personnalisés peut atteindre une longueur de 32767 octets. Par défaut, les valeurs sont tronquées à 256 octets. Saisissez une autre valeur pour modifier cette limite.

Fonctions supplémentaires

Cette boîte de dialogue génère une syntaxe pour la commande d'extension GATHERMD. Pour afficher l'aide de cette commande, exécutez la syntaxe suivante :

GATHERMD /HELP.

Dans la syntaxe, une liste des répertoires peut être spécifiée.

Conditions requises

Cette boîte de dialogue requiert le plug-in d'intégration pour Python et la commande d'extension GATHERMD. Pour IBM SPSS Statistics 19 et versions ultérieures, le plug-in et la commande d'extension sont installés avec le pack Essentials for Python. Pour plus d'informations, reportez-vous à Comment obtenir des plug-in d'intégration dans Système central > Questions fréquemment posées dans le système d'aide d'IBM SPSS Statistics. Si vous avez téléchargé la commande d'extension GATHERMD depuis la communauté SPSS, suivez les instructions dans le fichier associé Lisezmoi.
