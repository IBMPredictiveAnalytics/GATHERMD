Varrer arquivos de dados para informações da variável

Usando esta caixa de diálogo, é possível criar um conjunto de dados contendo informações sobre os dicionários de variáveis em um ou mais arquivos IBM® SPSS® Statistics, SA ou Stata.

►Para executar este procedimento, nos menus, escolha:

Arquivos
  Coletar Informações de Variável...

► Todas as variáveis nos arquivos selecionados estão incluídas.

Diretório para Procurar. Insira um diretório para procurar. Os respectivos subdiretórios também são pesquisados.

Padrão de Nome do Arquivo. Um padrão de expressão regular pode ser usado para filtrar os arquivos processados. Por exemplo, car limitaria os arquivos aqueles cujos nomes começam com "car". O padrão .*car aceitaria nomes de arquivos contendo "car".

• Expressões regulares não são o mesmo que curingas de nome de arquivo localizados em vários sistemas operacionais. Por exemplo, abc* faria a correspondência de qualquer nome começando com ab: isso significa literalmente ab seguido por zero ou mais c's.

• A expressão regular não faz distinção entre maiúsculas e minúsculas e é aplicada ao nome do arquivo sem a extensão.

Para obter uma explicação completa sobre expressões regulares, uma boa fonte é o endereço http://www.amk.ca/python/howto/regex/. Se não for fornecido nenhum padrão, todos os arquivos do tipo especificado serão processados.

Tipos de Arquivo para Varrer. Marque as caixas para os tipos de arquivo desejados. Os tipos de arquivo são:

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 e .xpt

• Stata: .dta

Lista de Atributos Customizados para Incluir. Insira os nomes de quaisquer atributos customizados cujos valores devem ser incluídos na saída.

Comprimento Máximo para Valores de Atributo. Valores de atributos customizados podem ter até 32767 bytes de comprimento. Por padrão, os valores são truncados em 256 bytes. Insira um valor diferente para alterar este limite.

Recursos adicionais

Esta caixa de diálogo gera sintaxe para o comando de extensão GATHERMD. Para exibir ajuda para este comando, execute a seguinte sintaxe:

GATHERMD /HELP.

Na sintaxe, é possível especificar uma lista de diretórios.

Requisitos

Esta caixa de diálogo exige o plug-in de integração para Python e o comando de extensão GATHERMD. Para o IBM SPSS Statistics 19 e posterior, o Plug-in e o comando de extensão são instalados com o pacote Essentials for Python. Para obter informações adicionais, consulte Como Obter Integration Plug-In, no Sistema Principal > Perguntas mais Frequentes no sistema de ajuda do IBM SPSS Statistics Help. Se você tiver feito download do comando de extensão GATHERMD da Comunidade SPSS, siga as instruções do arquivo Leia-me associado.
