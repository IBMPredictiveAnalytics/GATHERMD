Esegui scansione dei file di dati alla ricerca di informazioni sulle variabili

Tramite questa finestra di dialogo è possibile creare un dataset contenente le informazioni sui dizionari delle variabili in uno o più file IBM® SPSS® Statistics, SAS o Stata.

►Per eseguire questa procedura, dai menu scegliere:

 File
  Raccogli informazioni variabili...

► Sono incluse tutte le variabili nei file selezionati.

Cartella in cui effettuare la ricerca. Immettere una cartella in cui effettuare la ricerca. La ricerca viene eseguita anche nelle sottocartelle.

Modello nome file. Un modello di espressione regolare può essere utilizzato per filtrare i file elaborati. Ad esempio, car limiterà i file a quelli il cui nome inizia con "car". Il modello .*car accetta qualsiasi nome file contenente "car".

• Le espressioni regolari non sono identiche ai caratteri jolly dei nomi file presenti in molti sistemi operativi. Ad esempio, abc* corrisponde a qualsiasi nome che inizia con ab: letteralmente significa ab seguito da zero o più c.

• L'espressione regolare non riconosce la differenza tra maiuscole e minuscole e si applica al nome del file senza estensione.

Per una spiegazione esaustiva delle espressioni regolari, fare riferimento alla pagina http://www.amk.ca/python/howto/regex/. Se non viene fornito alcun modello, vengono elaborati tutti i file dei tipi specificati.

Tipi di file di cui eseguire la scansione. Selezionare le caselle relative ai tipi di file desiderati. I tipi di file sono:

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 e .xpt

• Stata: .dta

Elenco degli attributi personalizzati da includere. Immettere i nomi degli attributi personalizzati i cui valori devono essere inclusi nell'output.

Lunghezza massima dei valori degli attributi. I valori degli attributi personalizzati possono contenere fino a 32767 byte. Per impostazione predefinita i valori vengono troncati a 256 byte. Immettere un valore differente per modificare questo limite.

Funzioni aggiuntive

Questa finestra di dialogo genera la sintassi del comando di estensione GATHERMD. Per visualizzare la guida per questo comando, utilizzare la sintassi riportata di seguito:

GATHERMD /HELP.

Nella sintassi è possibile specificare un elenco di cartelle.

Requisiti

Questa finestra di dialogo richiede Integration Plug-In for Python e il comando di estensione GATHERMD. Per IBM SPSS Statistics 19 e versioni successive, il plug-in e il comando di estensione vengono installati con il package Essentials for Python. Per ulteriori informazioni, consultare Come ottenere i plug-in di integrazione in Sistema principale > Domande frequenti nella guida in linea di IBM SPSS Statistics. Se il comando di estensione GATHERMD è stato scaricato dalla Comunità SPSS, seguire le istruzioni nel file readme associato.
