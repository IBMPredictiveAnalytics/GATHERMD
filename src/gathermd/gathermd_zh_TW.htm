掃描資料檔以取得變數資訊

您可以使用此對話框建立資料集，其中包含一或多個 IBM® SPSS® Statistics、SAS 或 Stata 檔案中的變數字典相關資訊。

►若要執行此程序，請從功能表選擇：

 檔案
  收集變數資訊...

► 包含所選檔案中的所有變數。

要搜尋的目錄。 輸入要搜尋的目錄。也會搜尋子目錄。

檔名型樣。 可使用正規表示式型樣來過濾處理的檔案。例如，car 可將檔案限制為名稱以 "car" 開頭的檔案。型樣 .*car 可接受包含 "car" 的任何檔名。

• 正規表示式與在許多作業系統中找到的檔名萬用字元不同。例如，abc* 將與以 ab 開頭的任何名稱相符：它表示 ab 之後有零個或更多個 c。

• 正規表示式不區分大小寫，並且適用於沒有副檔名的檔案名稱。

如需正規表示式的完整說明，請參閱 http://www.amk.ca/python/howto/regex/。如果沒有提供任何型樣，則會處理指定類型的所有檔案。

要掃描的檔案類型。 勾選所需檔案類型的方框。檔案類型為：

• IBM SPSS Statistics：.sav、.por

• SAS：.sas7bdat、.sd7、.sd2、.ssd01 和 .xpt

• Stata：.dta

要包含的自訂屬性清單。 輸入要將其值包含在輸出中的任何自訂屬性名稱。

屬性值的最大長度。 自訂屬性值長度最多可以是 32767 個位元組。依預設會將值截斷為 256 個位元組。輸入不同的值以變更此限制。

其他特性

此對話框會產生適用於 GATHERMD 延伸指令的語法。若要顯示此指令的說明，請執行下列語法：

GATHERMD /HELP。

在語法中，可指定字典清單。

需求

此對話框需要 Integration Plug-In for Python 及 GATHERMD 延伸指令。對於 IBM SPSS Statistics 19 及更高版本，Plug-In 及延伸指令會隨 Essentials for Python 套件一起安裝。如需相關資訊，請參閱 IBM SPSS Statistics 說明系統中「核心系統」>「常見問題」下的「如何取得 Integration Plug-In」。如果您已從 SPSS 社群下載 GATHERMD 延伸指令，請遵循相關 Readme 檔中的指示。
