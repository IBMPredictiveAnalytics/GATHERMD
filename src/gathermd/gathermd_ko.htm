변수 정보에 대한 데이터 스캔 파일

이 대화 상자를 사용하면 하나 이상의 IBM® SPSS® Statistics, SAS 또는 Stata 파일에서 변수 사전에 대한 정보가 포함된 데이터 세트를 작성할 수 있습니다.

►이 프로시저를 실행하려면 메뉴에서 다음을 선택합니다.

 파일
  변수 정보 수집...

► 선택된 파일의 모든 변수가 포함됩니다.

검색할 디렉토리. 검색할 디렉토리를 입력하십시오. 해당 서브디렉토리도 검색됩니다.

파일 이름 패턴. 처리된 파일을 필터하기 위해 정규식 패턴을 사용할 수 있습니다. 예를 들어, car은 파일 이름이 "car"로 시작되도록 파일을 제한할 수 있습니다. .*car 패턴은 "car"이 포함된 모든 파일 이름을 허용합니다.

• 정규식은 많은 운영 체제에서 발견된 파일 이름 와일드카드와 동일하지 않습니다. 예를 들어, abc*는 ab로 시작하는 모든 이름과 매치합니다. 이는 ab 뒤에 0 또는 c가 옴을 의미합니다.

• 정규식은 대소문자를 구분하지 않으며, 확장자 없이 파일 이름에 적용됩니다.

정규식의 전체 설명을 보려면 http://www.amk.ca/python/howto/regex/를 참조하십시오. 패턴이 제공되지 않으면 지정된 유형의 모든 파일이 처리됩니다.

스캔할 파일 유형. 원하는 파일 유형에 해당하는 상자를 선택하십시오. 파일 유형은 다음과 같습니다.

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 및 .xpt

• Stata: .dta

포함할 사용자 정의 속성 목록. 값이 출력에 포함되어야 하는 사용자 정의 속성의 이름을 입력하십시오.

속성 값에 대한 최대 길이. 사용자 정의 속성은 최대 32767바이트가 될 수 있습니다. 기본적으로 값은 256바이트에서 잘립니다. 이 한계를 변경하려면 다른 값을 입력하십시오.

추가 기능

이 대화 상자는 GATHERMD 확장 명령에 대한 명령문을 생성합니다. 이 명령에 대한 도움말을 표시하려면 다음 구문을 실행하십시오.

GATHERMD /HELP.

명령문에서는 디렉토리 목록을 지정할 수 있습니다.

요구 사항

이 대화 상자를 사용하려면 Integration Plug-In for Python 및 GATHERMD 확장 명령문이 필요합니다. IBM SPSS Statistics 19 이상의 경우 이 플러그인과 확장 명령은 Essentials for Python 패키지와 함께 설치됩니다. 자세한 정보는 IBM SPSS Statistics 도움말 시스템의 Core System > 자주 묻는 질문에서 통합 플러그인을 얻는 방법을 참조하십시오. SPSS 커뮤니티에서 GATHERMD 확장 명령문을 다운로드한 경우 관련된 ReadMe 파일의 지침을 따르십시오.
