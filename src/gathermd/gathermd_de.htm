Datendateien nach Variableninformationen durchsuchen

Mit diesem Dialogfeld können Sie ein Dataset erstellen, das Informationen zu den Variablenwörterbüchern in mindestens einer IBM® SPSS® Statistics-, SAS- oder Stata-Datei enthält.

►Wählen Sie zur Ausführung dieser Prozedur folgende Optionen aus den Menüs aus:

Dateien
Variableninformationen sammeln...

► Alle Variablen in den ausgewählten Dateien werden eingeschlossen.

Zu durchsuchendes Verzeichnis. Geben Sie das zu durchsuchende Verzeichnis ein. Die zugehörigen Unterverzeichnisse werden ebenfalls durchsucht.

Dateinamensmuster. Das Muster eines regulären Ausdrucks kann verwendet werden, um die verarbeiteten Dateien zu filtern. Mit auto beispielsweise werden nur die Dateien ausgegeben, deren Namen mit "auto" beginnt. Bei dem Muster .*auto sind alle Dateinamen zulässig, die "auto" enthalten.

• Reguläre Ausdrücke sind nicht dasselbe wie Platzhalterzeichen für Dateinamen, die in vielen Betriebssystemen verwendet werden. So werden beispielsweise mit abc* alle Namen gefunden, die mit ab beginnen:

• Bei regulären Ausdrücken wird die Groß- und Kleinschreibung nicht beachtet und sie werden auf den Dateinamen ohne die Erweiterung angewendet.

Eine nützliche Quelle für eine vollständige Erläuterung regulärer Ausdrücke (in englischer Sprache) ist http://www.amk.ca/python/howto/regex/. Wenn kein Muster angegeben ist, werden alle Dateien mit den angegebenen Typen verarbeitet.

Zu durchsuchende Dateitypen. Wählen Sie die Kontrollkästchen für die gewünschten Dateitypen aus. Es handelt sich um folgende Dateitypen:

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 und .xpt

• Stata: .dta

Liste der einzuschließenden benutzerdefinierten Attribute. Geben Sie die Namen aller benutzerdefinierten Attribute ein, deren Werte in die Ausgabe eingeschlossen werden sollen.

Maximale Länge für Attributwerte. Benutzerdefinierte Attributwerte können bis zu 32.767 Byte lang sein. Werte werden standardmäßig auf 256 Byte gekürzt. Geben Sie einen anderen Wert ein, um diesen Grenzwert zu ändern.

Weitere Funktionen

In diesem Dialogfeld wird Syntax für den Erweiterungsbefehl GATHERMD generiert. Führen Sie folgende Syntax aus, um Hilfe für diesen Befehl anzuzeigen:

GATHERMD /HELP.

In der Syntax kann eine Liste der Verzeichnisse angegeben werden.

Anforderungen

Für dieses Dialogfeld sind das Integrations Plug-in for Python und der Erweiterungsbefehl GATHERMD erforderlich. Bei IBM SPSS Statistics 19 und höher werden das Plug-in und der Erweiterungsbefehl zusammen mit dem Paket "Essentials for Python" installiert. Weitere Informationen finden Sie im Hilfesystem zu IBM SPSS Statistics unter "Core-System > Häufig gestellte Fragen" im Thema "Beziehen von Integrations-Plug-ins". Wenn Sie den Erweiterungsbefehl GATHERMD von SPSS Community heruntergeladen haben, befolgen Sie bitte die Anweisungen in der zugehörigen Readme-Datei.
