Explorar archivos de datos buscando información de variables

Con este cuadro de diálogo puede crear un conjunto de datos que contenga información sobre los diccionarios de variables de uno o más archivos de IBM® SPSS® Statistics, SAS o Stata.

►Para ejecutar este procedimiento, desde los menús seleccione:

Archivos
  Recopilar información sobre variables...

► Se incluyen todas las variables de los archivos seleccionados.

Directorio de búsqueda. Especifique un directorio donde desee buscar. Se incluyen también los subdirectorios.

Patrón de nombre de archivo. Se puede utilizar un patrón de expresión regular para filtrar los archivos procesados. Por ejemplo, si especifica car se limitaría la búsqueda a los archivos cuyo nombre empiece por "car". El patrón .*car acepta los nombres de archivo que contengan "car".

• Las expresiones regulares no son lo mismo que los comodines para nombres de archivo que encontramos en muchos sistemas operativos. Por ejemplo, abc* coincide con cualquier nombre que empiece por ab: significa literalmente ab seguido de cero o más letras c.

• Una expresión regular no es sensible a mayúsculas y minúsculas y se aplica al nombre de archivo sin la extensión.

Para obtener una explicación completa de las expresiones regulares, se recomienda consultar http://www.amk.ca/python/howto/regex/. Si no se especifica ningún patrón, se procesan todos los archivos de los tipos especificados.

Tipos de archivos a explorar. Marque las casillas de los tipos de archivo deseados. Los tipos de archivo disponibles son:

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 y .xpt

• Stata: .dta

Lista de atributos personalizados que se incluirán. Introduzca los nombres de los atributos personalizados cuyos valores desea incluir en los resultados.

Longitud máxima de los valores de los atributos. Los valores de los atributos personalizados pueden tener hasta 32767 bytes de longitud. De forma predeterminada, los valores se truncan a 256 bytes. Especifique un valor distinto para cambiar este límite.

Características adicionales

Este diálogo genera sintaxis para el comando de extensión GATHERMD. Para mostrar la ayuda de este comando, ejecute la siguiente sintaxis:

GATHERMD /HELP.

En sintaxis, se puede especificar una lista de directorios.

Requisitos

Este cuadro de diálogo requiere el complemento de integración para Python y el comando de extensión GATHERMD. Para IBM SPSS Statistics 19 y versiones posteriores, el complemento y el comando de extensión se instalan con el paquete Essentials for Python. Si desea más información, consulte Cómo obtener complementos de integración, en Sistema principal > Preguntas más frecuentes en el sistema de ayuda de IBM SPSS Statistics. Si ha descargado el comando de extensión GATHERMD de la Comunidad de SPSS, siga las instrucciones del archivo léeme asociado.
