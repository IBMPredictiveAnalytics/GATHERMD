Просмотр файлов данных для поиска информации о переменных

Это диалоговое окно служит для создания набора данных с информацией о словарях переменных в одном или нескольких файлах IBM® SPSS® Statistics, SAS или Stata.

►Чтобы запустить эту процедуру, выберите в меню:

 Файлы
  Сбор информации о переменных...

► Затрагиваются все переменные в выбранных файлах.

Каталог для поиска. Введите каталог для поиска. Поиск будет выполняться также в его подкаталогах.

Шаблон имени файла. Для фильтрации обрабатываемых файлов может использоваться паттерн регулярного выражения. Например, car ограничивает поиск файлами, чье имя начинается на "car". Шаблон .*car позволяет принимать любые имена файлов, содержащие строку "car".

• Регулярные выражения не идентичны символам подстановки имени файла, известным во многих операционных системах. Например, abc* соответствует любому имени, которое начинается на ab: буквально это означает ab, за которым следует 0 или более "c".

• Регулярное выражение регистронезависимо и применяется к имени файла без расширения.

Хорошим источником, содержащим полное объяснение регулярных выражений, является http://www.amk.ca/python/howto/regex/. Если паттерн не задан, будут обрабатываться все файлы указанных типов.

Типы файлов для просмотра. Включите переключатели для нужных типов файлов. Возможные типы файлов:

• IBM SPSS Statistics: .sav, .por

• SAS: .sas7bdat, .sd7, .sd2, .ssd01 и .xpt

• Stata: .dta

Список пользовательских атрибутов для включения. Введите имена всех пользовательских атрибутов, значения которых нужно включить в вывод.

Максимальная длина для значений атрибутов. Длина значения пользовательского атрибута должна не превышать 32767 байт. По умолчанию значения усекаются до 256 байт. Чтобы изменить этот предел, введите другое значение.

Дополнительные возможности

Это диалоговое окно формирует синтаксис для команды расширения GATHERMD. Чтобы вывести справку для этой команды, введите следующую команду:

GATHERMD /HELP.

Этот синтаксис позволяет задать список каталогов.

Технические требования

Для этого диалогового окна требуется надстройка интеграции для Python и команда расширения GATHERMD. Для IBM SPSS Statistics 19 и выше эти надстройка и команда расширения устанавливаются вместе с пакетом Essentials for Python. Дополнительные сведения смотрите в разделе How to Get Integration Plug-Ins (Получение подключаемых модулей интеграции). Этот раздел находится в системе справки IBM SPSS Statistics: Core System > Frequently Asked Questions (Основная система > Часто задаваемые вопросы). Если вы скачали команду расширения GATHERMD с SPSS Community, то следуйте инструкциям в приложенном файле readme.
