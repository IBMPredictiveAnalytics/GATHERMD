変数情報のデータ ファイルをスキャンする

このダイアログ ボックスを使用して、1 または複数の IBMR SPSSR Statistics、SAS、または Stata ファイルの変数ディクショナリに関する情報を含んだデータセットを作成できます。

►この手順を実行するには、メニューから次のコマンドをを選択します。

 ファイル
変数情報を収集

► 選択したファイルのすべての変数が含まれます。

検索するディレクトリ: 検索するディレクトリを入力します。サブディレクトリも検索されます。

ファイル名のパターン: 正規表現のパターンを使用して、処理するファイルをフィルタリングできます。たとえば、car は、ファイルを名前が「car」で始まるファイルに制限します。パターン .*car は、「car」を含むファイル名を受け入れます。

• 正規表現は、多くのオペレーティング システムのファイル名のワイルドカードと同じではありません。たとえば、abc* は ab で始まるどんな名前にも一致します。ab の後に 0 または c が続いてもかまいません。

• 正規表現は大文字と小文字を区別しません。拡張子を除いたファイルの名前に適用されます。

正規表現の詳細な説明については、役立つ資料の 1 つとして http://www.amk.ca/python/howto/regex/ があります。パターンが指定されていない場合、指定した種類のすべてのファイルが処理されます。

スキャンするファイル形式: 該当するファイル形式のボックスをオンにします。ファイル形式は次のとおりです。

• IBM SPSS Statistics: .sav、.por

• SAS:.sas7bdat、.sd7、.sd2、.ssd01、および .xpt

• Stata: .dta

含めるカスタム属性のリスト: 出力に含まれるカスタム属性の名前を入力します。

属性値の最大長: カスタム属性値の長さは最大 32767 バイトです。デフォルトでは、256 バイトで切り捨てられます。この制限値を変更するには、異なる値を入力してください。

その他の機能

このダイアログは、GATHERMD 拡張コマンドのシンタックスを生成します。このコマンドのヘルプを表示するには、次のシンタックスを実行します。

GATHERMD /HELP

シンタックスでは、辞書のリストを指定できます。

要件

このダイアログには Integration Plug-In for Python と GATHERMD 拡張コマンドが必要です。IBM SPSS Statistics 19 以上の場合、Plug-in と拡張コマンドが Essentials for Python パッケージとともにインストールされます。詳しくは、IBM SPSS Statistics ヘルプ システムの「コア システム」>「よくある質問」の『Integration Plug-In の取得方法』を参照してください。SPSS コミュニティーから GATHERMD 拡張コマンドをダウンロードした場合、関連する readme ファイルの指示に従ってください。
